## Tech Used
* MVVM
* Firebase Auth & Firestore
* Custom TableView & CollectionView & NavigationController & TabBar
* JGProgressUHD
* Notification Banner
* Network Monitor 
* Programmatic Constraints 
* UIKit with no Storyboard
